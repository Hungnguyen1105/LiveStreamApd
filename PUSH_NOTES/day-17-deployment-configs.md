Day 17 - Deployment configurations

Mục tiêu:
- Thêm cấu hình deployment cho môi trường: dev/staging/production
- Thêm file cấu hình sample (application-dev.yml, application-prod.yml)
- Thêm hướng dẫn cơ bản để deploy (Docker, Heroku, Kubernetes manifest) nếu cần

Các file liên quan (gợi ý):
- src/main/resources/application-dev.yml
- src/main/resources/application-prod.yml
- Dockerfile (nếu cần)
- k8s/deployment.yaml (nếu cần)

Ghi chú: Đây là file ghi chú placeholder; nếu bạn muốn, tôi có thể tạo Dockerfile và manifest mẫu.
